# Progressive Stress Scenario v2

This scenario is an extended stress case for algorithm discrimination. It
reuses the official reconstructed PTIN data and changes only the disaster
overlay. The original four-edge official case remains available under
`data/scenario_reconstruction_progressive_v1`.

The eight failed PDN switches are selected from the official PDN topology by
combining restored load, critical-load weight, topology vulnerability exposure
and switch-controllability dependencies.
